# Investia
Projeto que integra frontend flutter com algoritmo de machine learning para ter recomendação de ações

# DSM-P5-G08-2025-2

Este projeto é composto por um sistema de recomendação financeira, dividido em backend (API Python) e front-end (aplicativo Flutter).

## Estrutura do Projeto
- **backend/**: API REST em Python para análise, recomendação e histórico financeiro.
- **front-end/**: Aplicativo Flutter para interação do usuário com o sistema.

## Como executar

### Backend
1. Acesse a pasta `backend`.
2. Instale as dependências: `pip install -r requirements.txt`
3. Execute: `python run.py`

### Front-end
1. Acesse a pasta `front-end/finance_ia`.
2. Instale dependências: `flutter pub get`
3. Execute: `flutter run`

## Funcionalidades
- Análise de perfil financeiro
- Recomendações personalizadas
- Histórico de ações
- Exportação de dados

## Requisitos
- Python 3.8+
- Flutter 3.0+
